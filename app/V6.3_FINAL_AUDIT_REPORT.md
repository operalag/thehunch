### 7.1 Trading Op Codes

| Operation | Op Code | Description |
|-----------|---------|-------------|
| Buy YES | `0x00000001` | Purchase YES position |
| Buy NO | `0x00000002` | Purchase NO position |
| Propose Answer | `0x00000003` | Submit resolution proposal |
| Challenge | `0x00000004` | Challenge proposed outcome |
| Settle | `0x00000005` | Finalize market resolution |
| Redeem | `0x00000006` | Claim winnings |
